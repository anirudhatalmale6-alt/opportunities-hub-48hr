== Description ==

A Funding & Institutions Hub for 48HoursReady.com featuring:
- Custom "Opportunities" post type
- Institution, Funding Type, Region, and Sector taxonomies
- Beautiful landing page with card-based listings and sticky filter bar
- Dedicated RSS feed at /feed/opportunities-feed
- AJAX-powered filtering (no page reload)

== Installation ==

1. Go to WordPress Admin > Plugins > Add New > Upload Plugin
2. Upload the opportunities-hub.zip file
3. Click "Activate"
4. Done! The plugin creates a "Funding Opportunities Hub" page automatically.

== How to Add Opportunities ==

1. Go to WordPress Admin > Opportunities Hub > Add New
2. Enter the opportunity title and description
3. Fill in the Details panel:
   - Funding Min/Max (dollar amounts)
   - Deadline date
   - Status (Open / Closing Soon / Closed)
   - Application URL (link to apply)
4. On the right sidebar, select:
   - Institution (e.g., IDB, World Bank)
   - Funding Type (e.g., Grant, Loan)
   - Region (e.g., Caribbean, Global)
   - Sector (e.g., SME, Agriculture)
5. Set a Featured Image (optional - shows as institution logo on the card)
6. Click Publish

== Managing Taxonomies ==

To add new Institutions, Types, Regions, or Sectors:
- Go to Opportunities Hub in the left menu
- Click on Institutions, Funding Types, Regions, or Sectors
- Add new terms as needed
